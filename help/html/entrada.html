Ventana de Entrada

En esta ventana se muestra una tabla con las partidas guardadas anteriormente. En la tabla se muestra la fecha de último guardado de cada partida, el nombre de jugador, el turno de juego por el que quedó la partida y el puesto en la clasificación entre el total de jugadores participantes.

Si se desea continuar una partida basta con seleccionarla de la lista y pulsar sobre "Cargar partida". Por el contrario, si ya no se quiere continuar una partida, puede optar por eliminarla. Para ello solo tienes que seleccionarla y pulsar sobre "Borrar partida".

La ventana también posee un cuadro para iniciar una partida nueva. En este caso, introduce un nombre de usuario que conste únicamente de letras del abecedario, mayúsculas y/o minúsculas, y que tenga una longitud entre 3 y 15 carácteres. Si tras cumplir los requisitos anteriores no existe una partida guardada con el mismo nombre, se iniciará una nueva partida con el nombre de jugador elegido.
